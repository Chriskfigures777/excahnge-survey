# excahnge-survey

The **Church Leader Survey** app lives in [`church-leader-survey/`](church-leader-survey/README.md). Full setup, scripts, and Airtable notes are documented there.

**Quick start** (from this repo root):

```bash
npm install --prefix church-leader-survey
cp church-leader-survey/env.example church-leader-survey/.env.local
# edit .env.local with your Airtable credentials
npm run dev
```

Or `cd church-leader-survey` and follow that folder’s README.
